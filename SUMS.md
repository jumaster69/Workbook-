## Resumen Ejecutivo

Este proyecto integra dos APIs públicas (DummyJSON y FakeStore API) en una aplicación Node.js con el objetivo de proporcionar ejemplos prácticos y documentación completa para consumir y procesar datos de APIs externas. El proyecto está diseñado como herramienta educativa para estudiantes del Politécnico Gran Colombiano.

## Requisitos del Proyecto

### Requisitos Funcionales

- Consumir productos de DummyJSON
- Consumir productos de FakeStore
- Buscar y filtrar productos
- Obtener detalles de productos
- Comparar precios entre APIs
- Generar estadísticas

### Requisitos No Funcionales

- Performance: Tiempo de respuesta menor a 500ms
- Disponibilidad: 99% de uptime de APIs
- Escalabilidad: Soportar 100+ requests simultáneos

- Confiabilidad: Manejo robusto de errores
