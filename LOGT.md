| Métrica | Valor | Fórmula |
|---------|-------|---------|
| **PV (Planned Value)** | 8.5 hrs | Valor planeado acumulado |
| **AC (Actual Cost)** | 13.5 hrs | Horas realmente gastadas |
| **EV (Earned Value)** | 11.0 hrs | Valor generado (~81% de tareas completadas) |
| **SV (Schedule Variance)** | +2.5 hrs | EV - PV (negativo = retraso) |
| **CV (Cost Variance)** | -2.5 hrs | EV - AC (negativo = sobrecosto) |
| **SPI (Schedule Performance Index)** | 1.29 | EV / PV (>1 = adelanto, <1 = retraso) |
| **CPI (Cost Performance Index)** | 0.81 | EV / AC (>1 = eficiente, <1 = costoso) |
